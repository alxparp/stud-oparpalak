# stud-oparpalak

This is an educational project aimed at gaining experience with maven modules, reflection cache, project structure, etc.

## Technologies

* Java 8
* Oracle DB
* H2DB
* Liquibase
* Hibernate 5
* Spring Framework 4
    - Spring MVC
    - Spring WebFlow
    - Spring Security
* JSF: Primefaces
* CSS
* SLF4J/LOG4J
* JUnit
* Maven
* Tomcat
